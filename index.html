About This Project

This study uses machine learning and satellite images to assess infrastructure development by analyzing factors like education, healthcare, farming, and economy. The model combines Random Forests and Convolutional Neural Networks (CNNs) to identify patterns in these factors and evaluate infrastructure levels. What makes this approach unique is that it visually maps infrastructure predictions onto satellite images, helping policymakers identify areas that need support. This method is faster and more cost-effective than traditional infrastructure surveys, which can be time-consuming and expensive. The CNN detects features such as agricultural land, buildings, and water bodies, which are then used to train another model that links these visual cues to infrastructure development. The goal is to provide better data for decision-making and improve resource allocation to support development more effectively.
